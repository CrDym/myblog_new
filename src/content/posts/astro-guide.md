Astro 是一个现代化的静态站点生成器，非常适合用来构建博客。

## 为什么选择 Astro

- **零 JS 默认** - 默认不发送任何 JavaScript，页面加载极快
- **内容集合** - 内置 Markdown 内容管理
- **灵活的框架支持** - 可以在组件中使用 React、Vue、Svelte 等

## 快速开始

创建一个新的 Astro 项目：

```bash
npm create astro@latest
```

然后安装依赖并启动开发服务器：

```bash
npm install
npm run dev
```

## 总结

Astro 是构建内容驱动网站的优秀选择。
